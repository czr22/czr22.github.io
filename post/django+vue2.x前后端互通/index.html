Django+Vue2项目

Jul 26, 2021
1 min read

一、创建基础框架

  1. 建立Django项目
$ django-admin startproject DjangoAndVue
  1. 进入DjangoAndVue项目
$ cd DjangoAndVue
  1. 建立Vue项目(使用vue ui可视化命令建立)
$ vue ui

二、 完善后端项目

主root: /DjangoAndVue/

  1. 建立后端子项目
$ django-admin startapp backend
  1. 修改连接数据库(sqlite3 -> mysql)
  • 进入/DjangoAndVue/setting.py
  • 修改DATABASES
DATABASES = {
        'default':{
            'ENGINE':'django.db.backends.mysql',#引擎名称
            'NAME':'django_mysql',#数据库名称
            'USER': 'root', # 用户名
            'PASSWORD': '12345678', # 密码
            'HOST': '127.0.0.1', # HOST
            'POST': 3306, # 端口

        }
}
  • 进入/DjangoAndVue/init.py添加引入mysql代码
import pymysql
pymysql.install_as_MySQLdb()
  • 生成迁移文件
$ python manage.py migrate
