Hugo基本操作 (Hugo中文文档)

  1. 创建 .md 文件
$ hugo new post/文件名称.md
  1. 生成public目录
根目录/myblog/
$ hugo --theme=主题名称 --baseUrl="http://czr22.github.io/"
注意，以上命令并不会生成草稿页面，如果未生成任何文章，请去掉文章头部的 draft=true 再重新生成。）
  1. 部署至github
$ cd public
$ git init
$ git remote add origin https://github.com/czr22/czr22.github.io.git
$ git add -A
$ git commit -m "first commit"
$ git push -u origin master
